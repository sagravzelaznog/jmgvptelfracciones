## 📜 Clasificación Principal de Fracciones ### 1. Fracciones Propias

Una fracción es **propia** cuando el **Numerador es menor que el Denominador** ($a < b$).

Este tipo de fracción siempre representa una cantidad **menor que la unidad** (menor que 1).

#### Ejemplo 1: Identificación (Paso a Paso)

Problema: Identificar si $\frac{5}{9}$ es propia.

Paso 1: Comparar el Numerador ($5$) con el Denominador ($9$).

Paso 2: Dado que $5 < 9$, el numerador es menor que el denominador.

Resultado: $\frac{5}{9}$ es una **Fracción Propia**.

#### Ejemplo 2: Representación Visual (Paso a Paso)

Problema: Representar $\frac{2}{3}$.

Paso 1: Se dibuja una sola unidad y se divide en 3 partes iguales (Denominador).

Paso 2: Se toman 2 de esas partes (Numerador).

Resultado: El total tomado es visiblemente menor que la unidad completa.

--- ### 2. Fracciones Impropias

Una fracción es **impropia** cuando el **Numerador es mayor o igual que el Denominador** ($a \ge b$).

Este tipo de fracción siempre representa una cantidad **igual o mayor que la unidad** (mayor o igual que 1).

#### Ejemplo 1: Identificación (Paso a Paso)

Problema: Identificar si $\frac{10}{7}$ es impropia.

Paso 1: Comparar el Numerador ($10$) con el Denominador ($7$).

Paso 2: Dado que $10 > 7$, el numerador es mayor que el denominador.

Resultado: $\frac{10}{7}$ es una **Fracción Impropia**.

--- ### 3. Números Mixtos

Un **Número Mixto** es la forma de expresar una fracción impropia. Consiste en un **número entero** y una **fracción propia**.

Fórmula: $\text{Entero} \frac{\text{Numerador}}{\text{Denominador}}$

#### Ejemplo 2: Conversión de Impropia a Mixta (Paso a Paso)

Problema: Convertir la fracción impropia $\frac{11}{4}$ a número mixto.

Paso 1: Dividir el numerador (11) entre el denominador (4): $11 \div 4 = 2$ y sobra $3$.

Paso 2: El **Cociente (2)** es el número entero.

Paso 3: El **Residuo (3)** es el nuevo numerador.

Paso 4: El **Divisor (4)** es el denominador (se mantiene).

Resultado: $2 \frac{3}{4}$ (Dos enteros y tres cuartos).

--- #### Ejemplo 3: Conversión de Mixto a Impropia (Paso a Paso)

Problema: Convertir el número mixto $3 \frac{1}{5}$ a fracción impropia.

Paso 1: Multiplicar el entero ($3$) por el denominador ($5$): $3 \times 5 = 15$.

Paso 2: Sumar el resultado al numerador ($1$): $15 + 1 = 16$. Este es el nuevo numerador.

Paso 3: El denominador se mantiene ($5$).

Resultado: $\frac{16}{5}$ (Dieciséis quintos).

--- ### 🧠 Actividades Sugeridas (Con Solución Oculta)

Actividad 1: Clasificación Múltiple. Clasifica las siguientes fracciones: $\frac{4}{5}$, $\frac{9}{8}$, $\frac{6}{6}$.

Ver Solución
  • $\frac{4}{5}$: **Propia** ($4 < 5$).
  • $\frac{9}{8}$: **Impropia** ($9 > 8$).
  • $\frac{6}{6}$: **Impropia** (porque es igual a la unidad, $6 = 6$).

Actividad 2: Conversión. Transforma el número mixto $4 \frac{2}{3}$ a fracción impropia.

Ver Solución

Fórmula: $(4 \times 3) + 2 = 14$. El denominador se mantiene $3$.

Resultado: $\frac{14}{3}$
